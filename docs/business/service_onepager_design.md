# サービス紹介資料 デザイン用構成案(表面・裏面)

> 用途: PowerPoint / Canva / Googleスライドで清書するためのレイアウト指示書。
> 文言は `service_onepager.md`(確定済み)を正とし、本書は「どこに・どう置くか」の
> ビジュアル設計に専念する。サイズは A4縦(または16:9スライド2枚)を想定。

## 共通デザインルール(ブランド統一)

既存サイト(souki-cp.co.jp)とトーンを揃える。営業現場で"同じ会社の資料"と
一目で分かることが信頼につながる。

| 項目 | 指定 |
|---|---|
| メインカラー | ブルー系(濃: `#1D4ED8` = blue-700 / 淡: `#EFF6FF` = blue-50) |
| アクセント | オレンジ系(`#F97316`)= 「初期費用0円」など"お得"の一点強調のみに使用 |
| 文字色 | 濃紺 `#1E3A8A`(見出し)/ ダークグレー `#374151`(本文) |
| フォント | 太めのゴシック(見出し=太字大、本文=中字)。Web版のInter相当 |
| 余白 | 詰め込みすぎない。1ブロック1メッセージ。行間は広めに |
| あしらい | 角丸カード(radius 16px)+ 薄い影。アイコンは線画(lucide相当) |
| トーン | 誇張しない。数字は大きく、но 断定的な効果保証は書かない |

- 初期費用0円は表面で1回だけオレンジ強調(多用すると安っぽくなる)

## 【裏面】= 納得(使い方→正直→会社→CTA)

目的: 「怪しくない・簡単・すぐ試せる」を伝え、行動(資料請求)まで運ぶ。

```
┌──────────────────────────────────────────────┐
│  [小見出し] 使い方は3ステップ(御社の作業は1日5分)  │
│   ①条件を伝える → ②毎朝メールが届く → ③検討する    │
│   [横並び・矢印でつなぐ・各ステップにアイコン]        │
├──────────────────────────────────────────────┤
│  [薄い枠のボックス] 正直にお伝えしていること         │
│   ・主に一般競争入札が対象(少額調達の一部は対象外)  │
│   ・情報の網羅性・正確性は公開情報に依存             │
│   ・応札の最終判断はお客様ご自身で                  │
│   ・相場データは過去実績に基づく参考値               │
│   [太字] 誇大な約束はしません。約束するのは          │
│          「探す手間ゼロ・毎朝お届け」だけ。          │
├──────────────────────────────────────────────┤
│  [会社の信頼・帯]                                 │
│   株式会社想樹「変化を力に変える伴走者」            │
│   補助金・公共入札・営業支援で中小企業を支援         │
├──────────────────────────────────────────────┤
│  [CTA・最下部・目立つ帯 blue-700]                 │
│   まずは無料で資料請求・トライアル                  │
│   御社の条件で「どんな案件が届くか」を無料でお試し    │
│   ┌──────────────┐                            │
│   │  QRコード      │  souki-cp.co.jp/nyusatsu-hisho│
│   └──────────────┘  d.senba@souki-cp.co.jp     │
│                      （連絡先はメールのみ）      │
├──────────────────────────────────────────────┤
│  [最下部・極小フォント] 出典・注記                  │
│   落札相場は調達ポータル(デジタル庁)落札実績         │
│   オープンデータに基づく参考値。料金・仕様は          │
│   予告なく変更する場合があります。                  │
└──────────────────────────────────────────────┘
```

**裏面の配置意図**:
- 「簡単さ(3ステップ)」で心理的ハードルを下げてから「正直な但し書き」を置く。
  先に不安を消し、後で誠実さを見せる順にする
- **CTAにQRコードを必須で入れる**。紙の資料から `/nyusatsu-hisho` LPへ直行させる
  (LP=フォーム=Formspreeの受信箱に着地。オンライン導線と一体化)
- 正直な但し書きは"弱み"ではなく"信頼の証明"として太字で締める

## 制作時のチェックリスト(清書担当者向け)

- [ ] 価格は必ず「月額3万円(税別)」「初期費用0円(期間限定)」で表記統一
- [ ] 「半額以下」「必ず落札」「全案件網羅」等の断定・誇大表現を使っていないか
- [ ] 比較表の大手料金に「各社公表情報に基づく目安」の注記があるか
- [ ] 出典表記(調達ポータル/デジタル庁)が裏面に入っているか
- [ ] QRコードのリンク先が `https://www.souki-cp.co.jp/nyusatsu-hisho` か
- [ ] 連絡先はメール(d.senba@souki-cp.co.jp)のみで、電話番号を記載していないか
- [ ] 色は青基調、オレンジは"お得"の一点のみ。多用しない
- [ ] 1枚に文字を詰めすぎていないか(余白で"高そう・信頼できそう"を演出)

## 展開案(この構成の使い回し)

同じレイアウト思想で以下に転用できる(文言は onepager から流用):
- **1枚PDF**(メール添付・DM同封用)
- **商談用スライド**(表面=1枚目、裏面=2枚目に分割)
- **LP との一貫性**: LPの見出し順とこの資料の順序を揃えてあるため、
  Web で見た人が資料をもらっても違和感がない(=同じ会社の一貫した体験)
