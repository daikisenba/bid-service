# 入札レコメンドサービス マーケティング基礎案

**前提**: 株式会社想樹の既存サービス(補助金コンサル・公共入札支援・AI営業支援)の
ラインに、本サービス「入札秘書」を追加する。ブランド軸「変化を力に変える伴走者」は
維持し、新サービスは既存サイト(souki-cp.co.jp)配下に1ページ追加する形が最適
(既存ドメインの信頼とSEO資産を活かせる。新規ドメインは作らない)。

**ターゲット**: NJSS等(月5.7万〜8.3万円)が高くて導入できない中小企業・個人事業主。
特に「全省庁統一資格を取ったのに案件を探せていない/探す時間がない」層。

**独自の売り(3点)**:
1. 大手より手の届く一律料金(月3万円・税別。NJSSは月5.7万円〜)。初期費用は今なら0円
2. 毎朝ゼロ手間(探しに行かなくても、条件に合う案件がメールで届く)
3. 相場が見える(公式落札実績データによる参考落札相場。下位競合には手薄)

---

## 1. キャッチコピー

**推奨(メイン)**:
> ## 探さない入札。毎朝、届く。
> 中小企業のための、月額3万円の「入札秘書」

代替案(A/Bテスト・媒体別に使い分け):

- 「入札案件を"探す時間"を、ゼロに。」(手間削減を前面に)
- 「NJSSは高い、と諦めた会社へ。」(競合比較・課題直撃。DM件名向き)
- 「資格は取った。あとは、案件が向こうから来るだけ。」(統一資格保有層に刺さる)
- 「毎朝5分。今日狙える入札が、メールで届く。」(具体的な体験を描写)

コピーの原則(この事業の北極星「規約クリーン=信頼」と整合):
- **誇大にしない。** 「全案件網羅」「必ず落札」は言わない(第3・4条の非保証と矛盾する)。
- 約束するのは「探す手間の削減」と「毎朝届く体験」。これは100%守れる。

## 2. ランディングページ(LP)構成案

既存サイトに追加する1ページの推奨構成(上から順に):

| # | セクション | 内容 |
|---|---|---|
| 1 | ヒーロー | キャッチコピー+サブコピー+CTAボタン(「無料で資料請求」/「トライアルを試す」) |
| 2 | 課題提起 | 「こんなお悩みありませんか?」案件探しに毎週何時間も/NJSSは高くて手が出ない/資格を取ったのに活かせていない |
| 3 | 解決(概要) | 条件を登録すれば、毎朝メールで候補が届く。1日5分で応札検討 |
| 4 | 3つの特徴 | ①毎朝自動でお届け ②参考落札相場つき ③月3万円(税別)・初期費用0円 |
| 5 | 料金比較 | NJSS等(月5.7万〜)vs 想樹(月3万円・税別)の正直な比較表。初期費用0円も訴求。※網羅範囲の違いも正直に明記 |
| 6 | 使い方3ステップ | ①条件を伝える→②毎朝メールが届く→③気になる案件を検討 |
| 7 | よくある質問 | 「全案件を網羅しますか?」→正直に「主に一般競争入札が対象。少額随意等は一部対象外」と回答。この正直さが信頼になる |
| 8 | 会社の信頼 | 想樹の公共入札支援の実績・伴走者としての姿勢。設立情報 |
| 9 | 最終CTA | 資料請求フォーム(会社名・担当者・メール・任意で興味のある品目) |

CTAは**「資料請求」を主、「トライアル」を副**に。いきなり契約ではなく、まず資料で
検討ハードルを下げる。フォームは項目を最小限(3〜4項目)に。

## 3. SEO戦略(検索で見つけてもらう)

### 狙うべき検索キーワード(検索意図別)

**競合比較・価格の意図(購入に最も近い・最優先)**:
- 「NJSS 高い」「NJSS 代替」「NJSS 比較 安い」「入札情報サービス 安い」
- 「入札情報サービス 比較 中小企業」「入札 情報 メール 通知 安い」

**課題・初心者の意図(潜在層。記事で受ける)**:
- 「入札 案件 探し方」「入札 参加 したい 中小企業」「官公需 入札 初めて」
- 「全省庁統一資格 案件 探す」「入札 案件 見つからない」
- 「オープンカウンタ 少額 調達 とは」

**地域×入札(ロングテール)**:
- 「東京 入札 情報」「(都道府県名) 入札 案件 探す」

### コンテンツ(お役立ち記事)アイデア

既存サイトにブログ/コラムを設け、以下を発信(SEO流入→LPへ誘導):

1. **「NJSSは高い?中小企業のための入札情報サービスの選び方」**
   → 競合比較の検索意図を丸ごと受ける。自社を選択肢の一つとして正直に位置づける
2. **「全省庁統一資格を取ったのに案件が見つからない人へ」**
   → 資格保有層(=見込み客そのもの)に直撃。統一資格の等級と案件の関係を解説
3. **「落札相場の調べ方 — 公式オープンデータの読み方」**
   → 自社の強み(相場データ)を教育コンテンツ化。専門性で信頼獲得
4. **「入札に初めて参加する中小企業が最初にやること」**
   → 潜在層の入口。官公需・電子入札の基礎を解説しCTAへ
5. **「オープンカウンタ(少額調達)とは — 中小企業が狙うべき理由」**

方針: **教育コンテンツで信頼を作り、比較記事で刈り取る。** 記事は自社の限界も
正直に書く(網羅性の非保証)。これがブランド「伴走者」と一貫し、離脱後の
クレームも防ぐ。

## 4. DM(ダイレクトメール)戦略

### 【最重要・先に読む】コンプライアンスの前提

AIで大量配信する前に、法規制を必ず押さえること。ここを外すと、この事業の
北極星「規約クリーン=信頼」を自ら破壊する。

- **特定電子メール法**: 広告・宣伝メールは原則**オプトイン(事前同意)**が必要。
  法人の公開アドレス宛など一部緩和はあるが、**送信者情報の明示**と
  **オプトアウト(配信停止)手段の明記**は必須。これを守らない一斉送信は違法リスク。
- **リストの入手方法**: 購入リスト・スクレイピングで集めたアドレスへの無差別送信は
  高リスク。**公的に取得できる情報からリストを作るのが最も安全**(下記)。
- 迷ったら、まずは**インバウンド(SEO・資料請求)を主軸**にし、DMは
  「正当に入手した連絡先」への少量・丁寧な配信から始める。

### クリーンなターゲットリストの作り方(この事業らしい強み)

調達ポータルが公開する**統一資格者情報のオープンデータ**(申請不要・公開)から、
「全省庁統一資格を持つ中小企業」のリストを合法的に作れる。これはまさに
「資格は持っているが案件を探せていない」見込み客そのもの。スクレイピングに頼らず
公式データでターゲティングできる点は、他社にない強み。
(※連絡先の入手・利用は各データの利用条件と特定電子メール法の範囲内で行うこと)

### DM文面のベース(郵送 or 正当に入手したメール向け)

**件名案(メールの場合)**:
- 「入札案件、毎朝お届けします(月3万円・初期費用0円)」
- 「【〇〇様】統一資格、活かせていますか?」

**本文ベース**:
```
株式会社〇〇
ご担当者様

突然のご連絡失礼いたします。株式会社想樹の千羽と申します。

全省庁統一資格をお持ちの企業様の中には、
「資格は取ったが、案件を探す時間がない」
「NJSSは高くて導入できない」
というお声を多くいただきます。

弊社の入札レコメンドサービスは、御社の条件に合った入札案件を
毎朝自動で探し、メールでお届けします。過去の落札相場もあわせてご覧いただけます。

・月額3万円(税別)、初期費用は今なら0円
・毎朝、条件に合う案件が届く
・過去の落札相場つき

まずは資料だけでもご覧いただけますと幸いです。
▼資料請求はこちら
(URL)

──────────────
株式会社想樹 千羽 太樹
(住所・電話・メール)
配信停止をご希望の場合はこちら:(オプトアウトURL/返信方法)
──────────────
```

文面の原則:
- **問題提起→共感→解決→軽いCTA(資料請求)**の順。いきなり売り込まない。
- 誇大表現を使わない(LP・規約と一貫)。
- **送信者情報とオプトアウトを必ず記載**(法令要件かつ信頼の証明)。
- AIで量産する場合も、会社名・担当者名を差し込む程度に留め、
  嘘や過度なパーソナライズ偽装はしない。

### 配信の進め方(段階的)

1. **フェーズA(今)**: インバウンド(LP+SEO記事)を立ち上げ、資料請求を受ける器を作る
2. **フェーズB**: 公式オープンデータ由来のリストへ、少量・オプトアウト付きで丁寧に配信し反応を検証
3. **フェーズC**: 反応の良かった文面・セグメントをAIで拡張(表示義務は維持)

## 5. 次アクション(優先順)

1. **LP 1ページを既存サイトに追加**(キャッチ+比較表+資料請求フォーム)。最速で作れる器
2. **資料(サービス紹介1枚 or 数枚)**を用意(DM・問い合わせ対応で使い回す)
3. **SEO記事を2本**(「NJSS 高い→選び方」「統一資格 活かせていない」)から着手
4. DMはフェーズA/Bの器が整ってから。先に売り込まない

> 補足: これらはマーケティングの叩き台であり、景品表示法(誇大広告の禁止)・
> 特定電子メール法の遵守を前提とする。特に料金比較・効果の表現は、事実に基づき
> 誇張しないこと(この事業の信頼設計と直結)。
