# サービス紹介資料 入稿用テキスト（コピペ完成版）

> 使い方: PowerPoint / Canva を開き、**A4縦を2枚**(または16:9スライド2枚)作成。
> 各ブロックの「▶ 貼るテキスト」をそのままコピペし、「配置・書式」の指示どおりに
> 置くだけで完成します。色コード・フォントサイズの目安まで指定済み。
>
> 料金表記は全ページ「月額3万円（税別）／初期費用0円（期間限定）」で統一。
> 誇大表現(半額以下・必ず落札・全案件網羅)は使わないこと。

## カラー / フォント早見表（最初にテーマ設定）

| 用途 | 色コード | サイズ目安(A4) |
|---|---|---|
| 見出し濃紺 | `#1E3A8A` | 大見出し 40pt / 中見出し 24pt |
| メインブルー(帯・強調) | `#1D4ED8` | — |
| 淡ブルー(背景・カード) | `#EFF6FF` | — |
| アクセント橙(0円のみ) | `#F97316` | — |
| 本文グレー | `#374151` | 本文 12〜14pt |
| 補足グレー | `#9CA3AF` | 注記 9〜10pt |

フォント: 見出し=太字ゴシック(Noto Sans JP Bold 等)/ 本文=同レギュラー。

# 【表面】A4-1枚目

## ① ヘッダー帯（最上部・全幅・背景 `#1D4ED8`・白文字）

▶ 貼るテキスト（左寄せ｜右寄せ）
```
株式会社想樹  ｜  入札レコメンドサービス
```
配置: 高さ8%程度の帯。左に社名(白・太字14pt)、右端に「入札レコメンドサービス」(白・11pt)。

## ② メインコピー（中央・大きく）

▶ 貼るテキスト（大見出し）
```
探さない入札。毎朝、届く。
```
▶ 貼るテキスト（サブ・一回り小さく）
```
中小企業のための、月額3万円の「入札秘書」
```
配置: 中央揃え。大見出し40pt `#1E3A8A` 太字。サブ18pt `#374151`。上下に余白を広めに。

## ③ 価格バッジ（横並び2枚・目立たせる）

▶ バッジ左（背景 `#EFF6FF`・角丸）
```
月額 3万円
（税別）
```
▶ バッジ右（枠線 `#F97316`・「0円」を橙で特大）
```
初期費用 0円
※期間限定
```
配置: 中央に2枚横並び。「3万円」「0円」の数字は28pt以上で最大化。右バッジの「0円」だけ `#F97316`。「※期間限定」は10pt補足グレー。

## ④ 悩み（チェックリスト）

▶ 貼るテキスト（小見出し）
```
こんなお悩み、ありませんか？
```
▶ 貼るテキスト（各行頭に ☑ アイコン）
```
☑ 資格は取ったが、案件を探す時間がない
☑ 入札情報サービスは高くて手が出ない（月5万円〜が相場）
☑ 毎回いろんなサイトを見て回るのが手間
☑ この案件、いくらで入れればいいのか相場が分からない
```
▶ 貼るテキスト（締め・太字・メインブルー）
```
→ その「探す時間」と「相場調べ」、まるごと自動化します。
```
配置: 小見出し16pt太字。チェック項目は13pt、2列×2行 or 1列4行。締めは14pt `#1D4ED8` 太字。

## ⑤ 3つの特徴（横3カラムのカード・背景 `#EFF6FF`）

▶ カード1（アイコン: ベル🔔）
```
毎朝、条件に合う案件が届く

業種・品目・地域・資格等級・価格帯を登録するだけ。平日の朝、条件に合う新着案件をメールでお届けします。
```
▶ カード2（アイコン: 右肩上がりグラフ📈）
```
過去の落札相場がわかる

公式の落札実績データに基づく「同種案件の落札相場（件数・中央値・価格帯・実例）」つき。応札価格の検討材料になります。
```
▶ カード3（アイコン: 盾🛡️）
```
月額3万円（税別）・初期費用0円

大手（月5.7万円〜）より手の届く一律料金。今なら初期導入費用は無料（期間限定）。無理なく続けられます。
```
配置: 3枚等幅。各カード上部にアイコン、タイトル14pt太字 `#1D4ED8`、本文11pt。

## ⑥ 料金比較表（表面の締め・最重要）

▶ 貼るテキスト（見出し）
```
料金比較
```
▶ 表（3列 × 6行。右列＝想樹列は背景 `#EFF6FF`・太字で強調）

| | 大手入札情報サービス | 想樹「入札秘書」 |
|---|---|---|
| 月額 | 5.7万円〜8.3万円 | 3万円（税別） |
| 初期費用 | 別途かかる場合あり | 今なら0円（期間限定） |
| 案件の探し方 | 自分で検索 | 毎朝メールで届く |
| 落札相場 | プランにより手薄 | 標準でお届け |
| 対象 | 全国・広範囲 | 一般競争入札を中心に厳選 |

▶ 貼るテキスト（表の下・極小注記）
```
※大手サービスの料金は各社公表情報に基づく一般的な目安です。
```
配置: ヘッダー行は `#1D4ED8` 白文字。想樹列(右)のセルは淡ブルー背景＋濃紺太字。注記9pt補足グレー。

# 【裏面】A4-2枚目

## ⑦ 使い方3ステップ（横並び・矢印でつなぐ）

▶ 貼るテキスト（見出し＋補足）
```
使い方は3ステップ
御社の作業は1日約5分。応札の判断に集中できます。
```
▶ ステップ（各ブロック：番号ラベル＋タイトル＋一言）
```
STEP 1  条件を伝える
業種・品目・地域・資格等級・価格帯をヒアリング（初回のみ）。

STEP 2  毎朝メールが届く
条件に合う新着案件が、相場つきで自動で届きます。

STEP 3  検討する
気になる案件だけ、専用シートで管理・応札を検討。
```
配置: 3ブロックを「①→②→③」と矢印(→)でつなぐ。STEPラベルは `#3B82F6` 太字、タイトル14pt濃紺。

## ⑧ 正直にお伝えしていること（薄い枠ボックス・信頼設計）

▶ 貼るテキスト（見出し）
```
正直にお伝えしていること（だから安心）
```
▶ 貼るテキスト（本文＋箇条書き）
```
本サービスは公的に公開されている情報を対象とするため、主に一般競争入札が対象です。ログインが必要な少額調達（オープンカウンタ等）の一部は自動探索の対象外です。

・情報の網羅性・正確性は、提供元（官公庁の公開情報）に依存します
・応札の可否・価格の最終判断は、お客様ご自身で行っていただきます
・相場データは過去の公開実績に基づく参考値です
```
▶ 貼るテキスト（締め・太字・濃紺）
```
誇大な約束はしません。お約束するのは「探す手間をゼロにし、毎朝候補をお届けする」こと。ここは確実に守ります。
```
配置: 全体を細枠 or 淡グレー背景のボックスに収める。本文11pt。締めのみ13pt太字 `#1E3A8A`。

## 仕上げチェック（出力前に確認）

- [ ] 料金は全て「月額3万円（税別）」「初期費用0円（期間限定）」で統一されているか
- [ ] 「半額以下」「必ず落札」「全案件網羅」等の誇大・断定表現がないか
- [ ] 料金比較表に「各社公表情報に基づく目安」の注記があるか
- [ ] 出典表記（調達ポータル/デジタル庁）が裏面最下部にあるか
- [ ] QRコードのリンク先が `https://www.souki-cp.co.jp/nyusatsu-hisho` か
- [ ] 連絡先はメール（info@souki-cp.co.jp）のみで、電話番号を記載していないか
- [ ] 橙色はバッジの「0円」1箇所のみに使い、多用していないか
- [ ] 文字を詰め込みすぎず、余白で"信頼感"が出ているか

## 10分で仕上げる手順（Canva想定）

1. 「A4 印刷用」テンプレートで2ページ作成
2. 上の早見表どおりにブランドカラー2色（濃紺・メインブルー）とアクセント橙を設定
3. 表面: ①→⑥を上から順にコピペして配置（比較表はCanvaの表要素を使用）
4. 裏面: ⑦→⑪を配置。QRはCanvaのQRアプリで生成しリンクを貼る
5. 仕上げチェックリストを確認 → PDFでダウンロード（印刷用・共有用の2種）
